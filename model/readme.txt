Для записи результатов измерений на SD-карте должен присутствовать
файл с именем ecodat.txt размером не менее 4 МБ (4 194 304 байт),
поскольку устройство не способно создавать файл и изменять его размер.

Желательно заранее заполнить файл ecodat.csv по шаблону.

Программа make_empty_file_4Mb создает вышеописанный файл для последующей записи на карту памяти.
Программа make_empty_sd_image создает виртуальный образ SD карты c таким файлом для отладки в эмуляторе Proteus.

Запись в файл осуществляется по кольцевому принципу: после достижения
конца файла следующая запись происходит в начале файла и заменяет старые.
